# 💰 Expense Tracker CLI App

A simple **Command Line Expense Tracker** developed in **Dart** as part of an academic assignment. The application demonstrates basic **Object-Oriented Programming (OOP)** concepts and allows users to manage expenses through a terminal-based interface.

## ✨ Features

* Add a new expense
* View all expenses
* Delete an expense
* Calculate the total expense
* Menu-driven command-line interface

## 🛠️ Built With

* Dart
* Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Prerequisites

* Dart SDK installed on your computer

Verify your installation:

```bash
dart --version
```

### Clone the Repository

```bash
git clone https://github.com/sharifahmedd/expense-tracker-cli-app.git
```

### Navigate to the Project

```bash
cd expense-tracker-cli-app
```

### Run the Program

```bash
dart run assignment_4.dart
```
